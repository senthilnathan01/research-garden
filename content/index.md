This site is a reusable **multi-project research garden** built on Quartz 4.

It is intentionally structured as a long-lived workspace rather than a one-off project site. Each project gets its own folder, navigation hub, and note clusters under `content/projects/<project-slug>/`.

- [[projects/index|Browse projects]]

- [[graph|View the graph]]
- [Return to the main site](https://senthilnathan01.github.io/)

## Garden principles

- Keep project content isolated by slug and folder.
- Use durable index pages for overviews, notes, papers, resources, and experiments.
- Add only real research over time. This starter contains placeholders, not fabricated findings.
- Reuse templates so new projects follow the same information architecture.

## Recommended structure

```text
content/
  projects/
    <project-slug>/
      index.md
      overview.md
      papers/index.md
      notes/index.md
      resources/index.md
      experiments/index.md
  templates/
```
