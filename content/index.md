This site is a reusable **multi-project research garden** built on Quartz 4.

It is intentionally structured as a long-lived workspace rather than a one-off project site. Each project gets its own folder, landing page, and a small set of working areas under `content/projects/<project-slug>/`.

## How this garden works

- Keep project content isolated by slug and folder so links stay stable over time.
- Use a project landing page plus a few high-signal working areas instead of many shallow pages.
- Use search, backlinks, and the graph together rather than relying on a single folder tree.
- Treat templates as reusable scaffolds, not published findings.
